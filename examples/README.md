# Example Configurations

This directory contains public-safe sample configurations that do not contain private paths, tokens, or machine-specific assumptions.

## Files

| File | Purpose |
|------|---------|
| `opencode-minimal.json` | Minimal opencode.json with relative plugin paths |
| `model-routing-policy-sample.yaml` | Sample advisory routing policy |
| `reviewer-trust-policy-sample.yaml` | Sample reviewer trust policy |

## Usage

Copy the relevant sample to your `.opencode/config/` directory and customize for your workspace.
